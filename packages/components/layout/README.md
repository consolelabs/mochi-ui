# @consolelabs/layout

Layout component is used for handling the overall layout of a page.

## Installation

Feel free to use the package manager of your choice:

```sh
# Using PNPM
pnpm i @consolelabs/layout

# Using NPM
npm i @consolelabs/layout

# Using Yarn
yarn add @consolelabs/layout
```
